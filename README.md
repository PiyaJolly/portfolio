# Personal Portfolio

My personal portfolio site, built from scratch with React and deployed on Vercel. It showcases my featured projects, background, and experience.

Live site: https://portfolio-six-sand-69.vercel.app

## About

I'm a final-year student at UTS completing a dual Bachelor of Engineering (Honours) in Data Science and Bachelor of Creative Intelligence and Innovation. My work spans computer vision, predictive modelling, full-stack development, and research, and this site brings those together in one place.

## Featured projects

- **NeuroVision**: an end-to-end deep learning system for brain tumour classification and segmentation, with Grad-CAM explainability and a full-stack interface. Repo: https://github.com/PiyaJolly/CNN-NeuroVision
- **Financial Market Pipeline**: a full-stack data pipeline with a rate-limited FastAPI backend and a React dashboard, deployed end to end. Live demo and code: https://github.com/PiyaJolly/financial-market-pipeline

## Tech stack

React, Vite, Tailwind CSS, and lucide-react for icons. Deployed on Vercel with automatic builds on every push.

## Running locally

```
npm install
npm run dev
```

The site then runs at http://localhost:5173.

## Structure

A single-page application with sections for Home, About, Projects, Experience, and Contact, organised as a tab-style navigation within one React component.
